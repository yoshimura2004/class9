도우미 신청 가이드

1. '도우미 신청' 메뉴 클릭

사이트 상단의 메인 메뉴에서 '도우미 신청'을 클릭하여 서비스 신청을 시작합니다.

2. 케어메이트 목록 확인

원하는 케어메이트의 등급(1급, 2급)을 선택하고, 등록된 케어메이트들의 목록을 확인합니다. 사이드바의 드롭다운 메뉴를 통해서도 등급을 선택할 수 있습니다.

3. 케어메이트 상세 정보 확인

목록에서 원하는 케어메이트를 클릭하면, 나이, 이름, 경력 등 상세 정보를 확인할 수 있는 팝업창이 나타납니다.

4. 도우미 신청 완료

상세 정보 팝업창에서 '도우미 신청' 버튼을 클릭하면, 신청이 성공적으로 완료되었다는 메시지 팝업이 나타납니다.

신청 내역은 '도우미 신청 확인'에서 확인할 수 있으며, 담당자가 빠른 시일 내에 연락을 드릴 것입니다!
